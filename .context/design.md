# Peeky 设计方案

## 产品定位

Peeky 是一个 macOS 菜单栏常驻应用，通过全局快捷键（默认长按 Command 键 1 秒）唤出全屏毛玻璃浮层，展示用户预设的备忘信息。核心价值是**零打断地快速获取高频易忘信息**。

## 技术栈

- Tauri v2 + Rust（基于 tauri-mac-starter 模板）
- React + TypeScript
- shadcn/ui + Tailwind CSS v4
- Jotai + TanStack Query
- SQLite（本地存储）

## 信息类型

- **纯文本片段**：常用命令、账号、地址、电话号码等零散信息
- **结构化对照表**：快捷键列表、代码 snippet、命令速查等表格形式

## 核心交互流程

### 浮层唤出

- 用户在任意应用下长按 Command 键超过 1 秒，屏幕覆盖全屏毛玻璃遮罩
- 遮罩上以多分栏布局展示备忘信息
- 浮层出现时带渐入动画（约 200ms）

### 浮层关闭

- 按 Esc 键关闭
- 再次按触发快捷键关闭
- 关闭时带渐出动画

### 触发方式

- 默认：长按 Command 键超过 1 秒
- 支持用户自定义触发方式（在设置中配置）

## 视觉设计

### 浮层风格

- 全屏毛玻璃遮罩，覆盖整个屏幕，背景模糊
- 信息以多分栏并排展示，每个分栏代表一个类别
- 风格对标 macOS 原生体验（类似 Spotlight 的毛玻璃质感）

### 布局

- 多分栏布局：多个类别分栏并排展示
- 每个分栏包含该类别下的所有条目
- 不做搜索/过滤功能，内容量可控，扫一眼即可

## 窗口架构

### 浮层窗口（Overlay Window）

- 全屏、无边框、透明背景
- 置顶于所有窗口之上
- 仅用于展示，不可编辑

### 管理窗口（Main Window）

- 标准 macOS 窗口
- 用于增删改分栏及条目
- 打开方式：
  - 点击菜单栏图标
  - 点击 Dock 图标
  - 自定义快捷键（如 Cmd+Shift+P，可配置）

## 系统存在形式

- Dock 栏显示图标
- 菜单栏常驻图标
- 支持快捷键打开管理窗口

## 数据存储

- 使用 SQLite 本地数据库
- 存储内容：分栏（类别）、条目、用户设置（触发方式、快捷键等）

## 何时更新本文件

- 产品目标、交互流程、视觉风格发生变化时
- 新增或移除核心功能时
- 数据结构或存储方案变化时
